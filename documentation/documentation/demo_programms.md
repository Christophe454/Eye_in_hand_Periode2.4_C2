# Voorbeeld programma's

Er is een voorbeeldprogramma beschikbaar voor het besturen van de robot door middel van de `movegroup`. De voorbeeld programma's kunnen dienen als templates voor je eigen software. 

:::::{card} 

::::{tab-set}

:::{tab-item} Python

Het Python-bestand van dit programma:
```bash
~/<workspace>/src/my_ur_ROS2/my_ur_demo/my_ur_demo/demo.py
```

Start het programma met:
```bash
ros2 run my_ur_demo demo
```

:::

:::{tab-item} Cpp

Het Cpp-bestand van dit programma:
```bash
~/<workspace>/src/my_ur_ROS2/my_ur_demo_cpp/src/demo.cpp
```

Start het programma met:
```bash
ros2 run my_ur_demo_cpp demo
```
